GoPlayMM is a free matchmaking system for Counter Strike: Global Offensive that aims to provide consistent 128tick servers, an easy to use platform, additional maps, in-depth stats to users, and additional features which enhance and build upon Valve's matchmaking concepts.

GoPlayMM is entirely web based and only requires a modern web browser, meaning NO third party downloads/clients/etc, which allows Mac and Wine users to join in on the fun!

Features

  • Fast, Consistent, Cross-Region 128tick servers

  • Easy to use, fast, simple, web-based Lobby/Queue System

  • No custom client; use vanilla Steam/CS:GO

  • Extensive list of Valve and Community maps

  • Wide range of in-depth statistics, achivements and graphs

  • Smart, skill-based ranking system

  • Multi-factor cheat/grief detection and prevention system

  • And a ton more!
